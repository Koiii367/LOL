## 丨工具介绍
* 解决更新过后的Vanguard反作弊引起的，不能通过修改快捷方式启动项更改语言的问题。
* 适用于英雄联盟国际服（欧服，美服等Riot运营的区服）和测试服（PBE）的简体中文汉化。

## 丨使用方法

* 1.下载 CnLoL 汉化工具。
* 2.确保退出LoL客户端和Riot启动器。
* 3.运行工具选择想要修改为简体中文的区服。
* 4.等待设置完成后启动客户端即可。

## 丨注意事项

* 如果修改失败，请以管理员身份运行以避免权限问题。
* 如提示Windows已保护你的电脑，请点击更多信息-仍要运行。
